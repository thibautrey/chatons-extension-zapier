Zapier Extension

This extension lets you interact with your Zapier account directly from Chatons.

Note: This extension is API-only and does not provide a graphical interface. All interactions happen through LLM tools in the conversation.

Features

  • List, create, enable, and disable Zaps
  • Browse available apps and their actions
  • Discover input/output fields and choices
  • Manage authentication connections
  • Browse Zap templates
  • AI-powered Zap suggestions (Guess a Zap)
  • Test individual Zap steps

Setup

To use this extension, you need a Zapier API key:

  1. Go to your Zapier Partner Settings to generate an API key
  2. The extension will prompt you for the key on first use
  3. Your key is stored locally and never shared
